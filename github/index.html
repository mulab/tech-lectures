$ git pull https://github.com/mulab/GitIntro.git

create a file named #{your_name}

$ echo hello>cgcgbcbc

run the command "git commit",it may look like:

$ git commit

#   （使用 "git add <file>..." 以包含要提交的内容）

$ git add .
$ git status
# 位于分支 master
# 要提交的变更：
#   （使用 "git reset HEAD <file>..." 撤出暂存区）

$ git status

nothing to commit, working directory clean

Best Practice

  1. pull before start working
  2. commit as often as possible
  3. push only after all your work is done
  4. *create your own working branch
  5. *rebase your branch whenever master branch goes ahead
  6. *open a pull request after your branch is done
